# Cgpygame
CGI study code made in python for students to practice the basics of geometric drawings, geometric transformations and projections using the pygame lib.

Requirements:

pygame 2.1.2 (SDL 2.0.18, Python 3.9.13 or 3.10)

Numpy 1.22.4
